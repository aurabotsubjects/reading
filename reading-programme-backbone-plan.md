## 2. The Weekly Rhythm (Mon–Fri, same shape every week)

One story = one week. Same 4-day skeleton every time so it becomes routine for students, freeing up their energy for the actual reading rather than "what do we do today."

| Day | Focus | Grouping | Curriculum emphasis |
|-----|-------|----------|---------------------|
| **Monday** | Lead-in + whole-class vocab & comprehension | Whole class | Oral language, vocabulary knowledge, activating prior knowledge |
| **Tuesday** | Guided reading | Level groups | Reading progression — decoding, fluency, comprehension at instructional level |
| **Wednesday** | Reading activities | Pairs | Comprehension strategies, collaborative talk, practice & knowledge |
| **Thursday** | Understanding project (part 1) | Pairs/small groups | Deeper comprehension, synthesis, creating a text/response |
| **Friday** | Understanding project (part 2) + Presentations | Pairs/small groups → whole class | Oral language, presenting ideas, key competencies (relating to others, participating & contributing) |

**Every lesson ≈ 50 minutes.** Suggested internal timing (adjustable per lesson type in the app):
- 5 min: settle + recap/hook
- 35–40 min: main activity
- 5–10 min: share-back / exit check

## 3. Day-by-Day Detail

### 🟦 Monday — Whole Class: Lead-in, Vocabulary & Comprehension
**Purpose:** Build the shared foundation everyone needs before independent/group work starts.

- **Lead-in question(s)** — 2–3 hook questions per story (predicting from title/cover image, connecting to students' own experience, activating background knowledge — e.g. "Have you ever moved somewhere new? What would you take?")
- **Vocabulary breakdown** — pull 6–10 key/tricky words from the story; explicit teaching (meaning, syllables, example sentence, connect to kupu Māori where relevant)
- **First read / read-aloud** — teacher reads or plays audio of the whole text; whole-class basic comprehension check (literal questions — who/what/where/when)
- **Curriculum link:** oral language component of structured literacy; vocabulary knowledge; activating prior knowledge as a comprehension strategy (NZC English Phase 2, Years 4–6)

### 🟩 Tuesday — Level Groups: Guided Reading
**Purpose:** Reading practice matched to instructional level, teacher rotates or runs one group at a time.

- Groups re-read the text (or a levelled excerpt) at their own pace/level
- Teacher-led group gets explicit decoding/fluency/comprehension instruction
- Other groups: independent re-read + simple task (e.g. sequence the events, find evidence for an answer)
- **Curriculum link:** differentiated instruction matched to reading progression; fluency and comprehension strategies at instructional level

### 🟨 Wednesday — Pairs: Reading Activities
**Purpose:** Active, hands-on comprehension work — this is where "fun" lives.

- Paired/hands-on activities such as: sequencing cards, vocabulary matching games, "hot seat" character/interviewee questioning, comprehension scavenger hunt, draw-and-label diagrams, true/false detective (find evidence in text), question each other using question cubes/dice
- **Curriculum link:** collaborative talk, comprehension strategies (inference, evidence-finding), practice consolidation

### 🟧 Thursday — Understanding Project (Part 1)
**Purpose:** Students synthesise their understanding into something they create — start the project.

- Pairs/small groups choose or are assigned a project format (rotate weekly for variety — see menu below)
- Planning + drafting/building time
- Teacher checks in with each group

### 🟥 Friday — Understanding Project (Part 2) + Presentations
**Purpose:** Finish, polish, and share.

- First ~20–25 min: finish project
- Remaining time: presentations to class (2–3 min per group) + peer feedback (e.g. 2 stars and a wish, thumbs poll)
- **Curriculum link:** oral language — presenting ideas clearly; key competencies — participating & contributing, relating to others

**Understanding Project menu (rotate through the term so it stays fresh):**
- Comic strip / storyboard retelling
- Diorama or model of a key scene/setting
- Podcast-style recorded interview (in role)
- Poster/infographic (non-fiction texts — facts, timeline, diagram)
- Short dramatic re-enactment / freeze-frame scenes
- "Fact file" or travel brochure (great fit for the voyaging/migration texts)
- Book trailer / pitch to sell the story to another class
- Debate or "would you have gone?" persuasive mini-speech

## 4. Curriculum Alignment (NZC English, Phase 2 — Years 4–6, effective from 2025 refresh)

The app should tag each lesson against these Phase 2 strands so you can show/print curriculum coverage per week:

1. **Reading progressions** — decoding, vocabulary, fluency, comprehension (literal, inferential, evaluative)
2. **Oral language** — spoken language development, structured talk, presenting
3. **Structured literacy components** — explicit, systematic, cumulative teaching of oral language + reading + writing
4. **Texts reflecting Aotearoa's bicultural heritage** — kupu Māori, Māori and Moriori narratives/perspectives (a strong fit given your story bank)
5. **Key competencies** — thinking; using language, symbols & text; managing self; relating to others; participating & contributing

Each week's lesson set (Mon–Fri) should show a short tick-list of which of these 5 areas it covers — this becomes your at-a-glance curriculum-coverage report for the term.

### Christian character reflection
Alongside the NZC strands, every week's full lesson plan includes **one reflection question per school value**, tied to that week's story:
- **Aroha** (Love: God, Others, Self, World)
- **Mōhio** (Know: God, His story and our place in it)
- **Manaaki** (Serve: faithfully with our gifts and talents)
- **Pānga** (Impact: bringing God's Kingdom to earth)

*Soli Deo Gloria — To God Alone the Glory*

## 5. App Structure (for the build phase)

### Core screens
1. **Term Overview** — grid of Week 1–11, each cell shows story title + status (planned/in progress/done), with an easy way to drag/reassign a story out to "next term" if the term runs shorter
2. **Week Planner** — the main workhorse screen
3. **Story Library** — list of the 11 stories with metadata (title, type, level, theme, uploaded file)
4. **Curriculum Coverage view** — auto-generated report of which Phase 2 strands have been covered, week by week

### Dropdowns (as requested)
On the **Week Planner** screen:
- **Dropdown 1: Week** → Week 1, Week 2 … Week 11
- **Dropdown 2: Day** → Monday / Tuesday / Wednesday / Thursday / Friday
- Selecting Week + Day loads that day's lesson: story reference, lead-in questions, vocab list, activity instructions, timing breakdown, curriculum tags — all editable/printable

Additional dropdown/selector ideas worth building in:
- **Story assignment** dropdown per week (pick which of the 11 stories runs that week — fully flexible, not locked to the table order)
- **Level group** selector (Tuesday) — set your own group names/levels once, reuse every week
- **Understanding project type** dropdown (Thu/Fri) — pick from the project menu above, or mark "custom"
- **Print/export** button — generate a one-page lesson summary for the day (handy to have on the whiteboard or printed for relievers)

### Data per story (what we'll build out next, story by story)
- Title, file link, type (fiction/non-fiction), reading level
- 2–3 lead-in questions
- Key vocabulary list (word + kid-friendly definition)
- Basic comprehension questions (literal, for Monday)
- Level-group discussion/task prompts (Tuesday)
- Paired activity instructions (Wednesday)
- Suggested understanding-project options for that specific story (Thu/Fri)
- Curriculum tags (which Phase 2 strands this story/week covers)
